# 🚀 Field Sales Automation & Order Tracking System

[![Platform](https://img.shields.io/badge/Platform-Android_Java-brightgreen.svg)](https://developer.android.com/)
[![Backend](https://img.shields.io/badge/Backend-Laravel_11_Sanctum-red.svg)](https://laravel.com/)
[![Database](https://img.shields.io/badge/Database-MySQL-blue.svg)](https://www.mysql.com/)

**Field Sales Automation** একটি কমপ্লিট সলিউশন যা মাঠ পর্যায়ের বিক্রয় কর্মীদের কার্যক্রম মনিটর, অর্ডার ম্যানেজমেন্ট এবং বকেয়া কালেকশনকে ডিজিটালাইজ করে। এটি একটি ইন্ডাস্ট্রি-গ্রেড অ্যাপ যা অফলাইন সিঙ্ক্রোনাইজেশন এবং রিয়েল-টাইম ট্র্যাকিং নিশ্চিত করে।

---

## 📅 The 20-Day Development Journey

### 🏗️ Phase 1: Foundation & Backend (Day 1 - 8)
- **Analysis:** বিজনেস প্রবলেম অ্যানালাইসিস এবং MVP ফিচার লিস্ট ফাইনাল করা।
- **DB Design:** UUID ব্যবহার করে স্কেলেবল ERD তৈরি (Users, Outlets, Products, Orders)।
- **Laravel Setup:** Laravel 11 এবং Sanctum ব্যবহার করে সিকিউর API আর্কিটেকচার।
- **Order Logic:** কমপ্লেক্স ক্যালকুলেশন ইঞ্জিন এবং ডাটা ইন্টিগ্রিটি নিশ্চিত করা।
- **GPS Engine:** ফিল্ড অফিসারদের রিয়েল-টাইম লোকেশন স্টোরেজ এবং রুট ট্র্যাকিং API।

### 📱 Phase 2: Android Development & UI/UX (Day 9 - 14)
- **Architecture:** Scalable MVVM আর্কিটেকচার এবং Retrofit নেটওয়ার্কিং সেটআপ।
- **Premium UI:** Lottie Animation এবং Material Design ব্যবহার করে লগইন, রেজিস্ট্রেশন ও প্রোডাক্ট ক্যাটালগ।
- **Admin Panel:** লারাভেল দিয়ে সেন্ট্রালাইজড ড্যাশবোর্ড (Live Search, CRUD, Image Handling)।
- **Shop Management:** অ্যাপ থেকে সরাসরি আউটলেট রেজিস্ট্রেশন এবং ডাটা সিঙ্ক্রোনাইজেশন।

### 🧾 Phase 3: Sales, Invoicing & Profile (Day 15 - 20)
- **Order System:** ইন-অ্যাপ মেমো ভিউ এবং এক ক্লিকে **WhatsApp**-এ ইনভয়েস শেয়ারিং।
- **Reporting:** ডুপ্লিকেট ডাটা রিমুভ করে নির্ভুল সেলস রিপোর্ট এবং সাব-কোয়েরি লজিক।
- **Due Management:** অটোমেটেড ডিউ ট্র্যাকিং এবং ইনস্ট্যান্ট কালেকশন সিঙ্ক সিস্টেম।
- **Profile Module:** Multipart API ব্যবহার করে প্রোফাইল পিকচার আপলোড এবং Android 13+ পারমিশন হ্যান্ডেল।

## ✨ Core Features

| Feature | Description |
| :--- | :--- |
| **🔐 Secure Auth** | Laravel Sanctum ভিত্তিক টোকেন ম্যানেজমেন্ট। |
| **📍 GPS Tracking** | সেলস অফিসারদের মুভমেন্ট এবং রুট ট্র্যাকিং। |
| **📦 Order Engine** | অফলাইন-টু-অনলাইন সিঙ্ক এবং ডিউ ক্যালকুলেশন। |
| **📄 Digital Invoice** | কাস্টম মেমো জেনারেশন এবং WhatsApp ইন্টিগ্রেশন। |
| **🖼️ Profile Sync** | ইমেজ আপলোড সহ ডাইনামিক ইউজার ম্যানেজমেন্ট। |

## 🛠️ Technology Stack

- **Android:** Java, MVVM, Retrofit, Glide, Lottie, Material Components.
- **Backend:** Laravel 11, Sanctum API, AJAX Live Search.
- **Database:** MySQL (Relational Schema with UUID).
- **Tools:** Postman (API Testing), Git (Version Control).
